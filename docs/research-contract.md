## Mission

This project exists to surface high-value interdisciplinary hypotheses and make them testable.

The system is not allowed to pretend that:

- a generated idea is already a discovery
- novelty is proven without prior-art review
- confidence equals truth
- a Nobel-class outcome can be guaranteed

## Non-Negotiable Rules

1. Verify before claim.
2. Separate fact, inference, hypothesis, and experiment.
3. Every scored hypothesis must include evidence provenance.
4. Every top-tier hypothesis must include a cheap falsification route.
5. Unknowns stay marked as unknowns until checked.

## Outcome Ladder

The project should optimize for the following order:

1. honest evidence synthesis
2. reproducible computational validation
3. publication-quality research package
4. external scientific review
5. downstream recognition if earned

## Nobel-Oriented Constraint

This repository may support work that could eventually contribute to Nobel-level research, but it must not claim:

- that the work already qualifies
- that self-nomination is possible
- that novelty for humanity has been proven from local retrieval alone

The operational target is a defensible `top-5` shortlist with validation artifacts, not award rhetoric.

## Top-5 Promotion Gate

A hypothesis can enter the `top-5` only if all conditions hold:

- evidence trail exists
- novelty is plausible and not obviously already mainstream
- Python-first validation can be executed or clearly staged
- first falsification test is explicit
- objections are listed
- validation cost is acceptable for the current phase

## Required Deliverable Shape

Each serious hypothesis must expose:

- title
- fields bridged
- core mechanism
- known facts
- inferred bridge
- speculative leap
- evidence source ids
- novelty, feasibility, falsifiability, impact, validation cost
- confidence and its basis
- minimal Python prototype
- open dataset or simulation route
- first falsification test
- main objections
