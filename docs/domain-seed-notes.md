# Domain Seed Notes

These notes summarize the initial domain framing used to bootstrap the engine.

They are internal planning inputs, not external scientific proof.

## Biology

Priority unresolved areas:

- aging mechanisms
- regulatory roles of poorly characterized genes
- regenerative control of cell populations
- robust interpretation of noisy high-dimensional biological data

Candidate transferable methods:

- graph learning
- control theory
- information-theoretic feature selection
- topological data analysis

## Materials Science

Priority unresolved areas:

- multiscale structure-property modeling
- phase stability in complex alloys
- defect and degradation prediction
- fast screening of large design spaces

Candidate transferable methods:

- information bottleneck
- graph learning
- active learning
- surrogate modeling

## Control And Information Theory

Priority unresolved areas:

- static output feedback
- robust control under uncertainty
- delayed and stochastic dynamics
- efficient reduced-order representations of complex systems

Candidate transferable methods:

- feedback design
- Kalman-style filtering
- robust control
- information-theoretic compression
